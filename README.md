# 💻 Logiciels utilisés

* Moteur de jeu : **Babylon.js**
* Modélisation : **Blender**
* IDE : **WebStorm**
* Musique : **MuseScore 4**
* Textures : **Procreate**
* Cinématiques : **DaVinci Resolve**
* Gestion de fichiers : **GitHub**
