# 💻 Logiciels utilisés

* Moteur de jeu : **Babylon.js**
* Modélisation : **Blender**
* IDE : **WebStorm**
* Musique : **Musecore 4**
* Textures : **Procreate**
* Cinématiques : **DaVinci Resolve**
* Gestion et stockage de fichiers : **GitHub**, **Dropbox**
